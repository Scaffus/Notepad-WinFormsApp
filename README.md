# Notepad-WinFormsApp
Notepad in c# .net

# Features

- Create, save, open note
- Undo, redo, copy, cut, paste, select all
- Change font and color
- Fulscreen mode
- Current column and line info
- Word count info
- Zoom percentage info
- App title is file name
- Close, new and open safety (popup message)
